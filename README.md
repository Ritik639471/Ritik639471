[![Frontend](https://img.shields.io/badge/Live_Frontend-Vercel-black?style=flat-square&logo=vercel)](https://aura-app.vercel.app)
[![Backend](https://img.shields.io/badge/Live_Backend-Render-46E3B7?style=flat-square&logo=render)](https://aura-app-keg8.onrender.com)

**Tech Stack:** `React 18` · `Node.js` · `Express` · `MongoDB` · `Socket.io` · `JWT` · `Cloudinary` · `Nodemailer`

| 💬 Real-Time Messaging | WebSocket-powered instant chat via Socket.io |

| 👥 Admin System | Room creators can promote/demote admins, pin & clear messages |
| 📎 Media Sharing | Image uploads via Cloudinary, URL link previews with OG metadata |
| ✏️ Message Actions | Edit, delete, react (6 emojis), read receipts, typing indicators |
| 🔐 Auth | JWT sessions, bcrypt passwords, OTP-based password reset via email |

### 🎬 [RM Movies — Full-Stack Movie Discovery App](https://github.com/Ritik639471/RM_Movies)

> A feature-rich MERN stack movie app powered by the **TMDB API**, with personalized watchlists, reviews & trailer playback.

**Tech Stack:** `React 19` · `Vite` · `Node.js` · `Express` · `MongoDB` · `JWT` · `Framer Motion` · `TMDB API`

| Feature | Details |
|---|---|
| 🔍 Search & Discover | Live TMDB search + trending feed with genre/year/sort filters |
| 🎯 Recommendations | Personalized "Recommended For You" based on your watchlist |
| 🎥 Trailer Playback | Embedded YouTube trailer inside a rich movie detail modal |
| 💾 Watchlist | Add/remove movies, track "Plan to Watch" vs "Watched" status |
| ⭐ Reviews & Ratings | Community star ratings + written reviews with upsert logic |
| 🔐 Auth | Custom JWT + bcrypt auth (no third-party services) |
| ✨ Animations | Framer Motion staggered reveals, spring transitions, hover effects |
| 🎨 UI | Dark glassmorphism, gradient text, glowing purple/pink accents |
